📖 Understanding FIFO (First-In-First-Out)

FIFO is the simplest page replacement algorithm used by operating systems to manage memory.

  • How it works: The OS tracks all pages in memory in a queue. When memory is full and a new page needs to be loaded (a Page Fault), the OS evicts the oldest page (the one that entered first) to make room.
  • Analogy: Think of a queue at a grocery store. The person who has been in line the longest gets served and leaves first.
  • The Catch (Belady's Anomaly): While simple, FIFO is not always optimal. It suffers from a strange phenomenon where giving the system more memory frames can sometimes cause more page faults!
